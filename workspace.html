Welcome to Interactive Workspace

Notepad (top-left) — Rich text editor with formatting, colors, and auto-save.

Whiteboard (top-right) — Draw with pen, shapes, text. Use Size and Opacity sliders to adjust.

Scratch Sheet (bottom-left) — Click any cell to edit. Supports formulas: =SUM(A1:A5) =AVG =SPARK

Analytics (bottom-right) — Stock charts with comparison view (click ➤ to compare), Concept Map, or Graph Calculator with shading & export.

Use the pane tabs in the header (All / Notes / Draw / Sheet / Map) to focus on one pane. Click the expand arrow in any pane header to maximize it. The Media button stores URLs and plays audio.
